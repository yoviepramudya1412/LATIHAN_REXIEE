Enter dalam html tidak menggukan enter biasanya seperti di microsoft. Jika menggunakan enter pada html, maka html tidak membacanya sebagai enter. Di html enter menggunakan tanda tag br. Disarankan jika ingin enter membuat peragraf baru. Tag br adalah void elemen, jadi tidak memiliki konten.


Ini adalah contoh tag br atau cara menggunakan enter di html.

Apa itu void elemen? void element adalah elememt pada html yang tidak memiliki simpul turunan apapun yaitu element bertumpuk atau simpul teks. element awal yang memiliki tag awal; tag akhir tidak boleh ditentukan untuk element kosong. Dalam html, void element tidak boleh menggunakan tag penutup.
