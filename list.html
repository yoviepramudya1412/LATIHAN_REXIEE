MEMBUAT LIST

List memiliki dua tipe, yaitu unordered list dan ordered list.

unodered list

untuk membuat daftar tulisan kita bisa menggunakan tag ul di html, tiap daftar isinya kita bisa menggunakan tag li. contohnya:

Daftar Hobi

  • tidur
  • musik

ordered list

jika ingin membuat daftar berturut memiliki angka kita bisa mengganti tag ul menadi tag ol. untuk daftar isinya tetap sama menggunakan tag.

Sebutkan bagian indra manusia

  2. hidung
  3. mulut
  4. telinga
  5. kulit
