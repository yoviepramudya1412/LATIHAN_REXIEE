Belajar favicon

  • Saat memebuka website biasanya di tab terdapat logo perusahaan seperti faceebook, google dan lainnya. Itu yg dinamakan favicon.
  • Untuk membuat favicon kita bisa menggunakan tag link pada bagian head.
  • diamana dalam link kita perlu menggunakan atribute rel="icon" dan href menuju lokaasi gambar icon.
  • gambar icon bisa file berupa ico, png, jpg, svg, atau gif.
