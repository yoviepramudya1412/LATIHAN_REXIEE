Belajar Link

  • html memiliki fitur link atau tautan, dimana kita bisa meng-klik link tersebut, dan berpindah ke halaman html lain.
  • Link tidak harus dalam bentuk teks, link juga bisa dalam bentuk gambar.

Tag a

  • Untuk membuat link di html , bisa menggunakan tag a.
  • isi konten tag a adalah isi dari tampilan link, bisa teks, atau yg lainnya.
  • tag a memiliki atribute href, yang berisikan lokasi tujuan link tersebut.
  • tag a juga memiliki atributte target, yg di gunakan sebagai target jendela web browser.
  • tag a juga memiliki atributte tittle, untuk menuliskan judul yg keluar ketika mouse berada diatas link tsb.
