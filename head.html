Belajar Head

  • Selain tag body, di HTML juga menggunakan tag head
  • Tag head digunakan untuk menambah informasi yang tidak dimunculkan dihalaman web
  • Head biasanya juga digunakan untuk menambahkan informasi seperti CSS atau Javascript
  • Head juga biasa digunakan untuk menambahkan metadata atau informasi tambahan, untuk halaman HTML, biasanya digunakan oleh Search Engine agar dimengerti oleh search engine seperti google.
  • contoh tag yang biasa digunakan di tag head adalah tag tittle, gunakan untuk mengubah judul halaman web browser.
